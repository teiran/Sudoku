# Sudoku

Tarkoituksena on tehdä sudokupeli jolla pystyy pelaamaan sudokua. Sudokua voi pelata, sudokussa on sekä valmiita vaikeita karttoja mutta myös generoituja karttoija, jotka muuttuvat jokakerta kun generointi nappulasta painaa.
